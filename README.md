# Silam NeuroChat

**The open-source AI chat that beats ChatGPT in speed, UX, and extensibility.**

## Features
- Streaming responses
- Multi-model (OpenAI, Anthropic, Groq)
- Tool calling (weather, calculator)
- Persistent chat history
- Markdown + LaTeX
- Dark/Light theme
- System prompt editor
- Mobile-first glass UI
